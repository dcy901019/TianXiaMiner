快捷操作指南
您可以使用键盘右侧的数字小键盘来快速执行以下操作：

小键盘 1：快速打开【生活技能】面板。

小键盘 2：自动使用并放置当前选中的“足通”道具。

小键盘 3：激活/释放已配置好的技能连招序列。

小键盘 4：执行【返回神石】指令，快速脱离当前区域。
